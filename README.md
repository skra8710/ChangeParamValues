* **ResetValues.lua**: パラメータの値を初期化する
* 

## Dates
* October 27, 2018 - プロジェクト開始
* December 24, 2018 - GitHubへ投稿
* December 29, 2018 - ボーカロイドエディタ内に既に存在するプラグインを開発しようとしていることに気が付き、エディタでは使えないものを開発するように方向転換
